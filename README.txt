Asteroids by Kevin Mendoza Tudares, Winter 2021, COMP_ENG 205
--------------------------------------------------------------------------------
To play the game, move the ship using the WASD keys, rotate the ship using the 
left and right arrow keys, fire using the space bar, and pause using the P key

Colliding with an asteroid will result in a loss

Win the game by destroying all 6 asteroids


Features
--------------------------------------------------------------------------------
Sound or Music
Gravity or Other Forces (objects fall/move with non-constant velocity)
Multiple in-flight projectiles
